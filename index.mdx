Dalamud is the plugin development framework that powers the FFXIV modding ecosystem. Injected into the game via [XIVLauncher](https://github.com/goatcorp/FFXIVQuickLauncher), it exposes a comprehensive set of services — game data, chat, UI rendering, function hooking, and more — that let you build powerful in-game plugins with C#.

## Get started

<Steps>
  <Step title="Install XIVLauncher">
    Download [XIVLauncher](https://goatcorp.github.io/) — it manages Dalamud installation and injection automatically. You do not need to install Dalamud manually.
  </Step>
  <Step title="Set up your development environment">
    Install the .NET 8 SDK and your preferred IDE (Visual Studio or Rider). See the [building guide](/building) for all required dependencies.
  </Step>
  <Step title="Create your plugin project">
    Follow the [Quickstart](/quickstart) to scaffold a new plugin using the official template and wire up your first service.
  </Step>
  <Step title="Explore the API">
    Browse the [API Reference](/api/plugin/idalamudplugin) to discover services for game state, UI, chat, hooking, and more.
  </Step>
</Steps>

<Note>
  If you just want to **use** Dalamud plugins as a player, download [XIVLauncher](https://goatcorp.github.io/) — you don't need to read these docs. This site is for **plugin developers**.
</Note>
